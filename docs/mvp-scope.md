# MVP 范围

豆小宝家长端 MVP 先做“安全、通信、设备、管控”的底座，再用家庭 Agent 把这些能力组织成家长能直接执行的工作流。

## P0 必做

### 工作台

- 孩子在线状态、电量、设备同步时间。
- 今日位置、今日任务、快捷呼叫、寻找手表。
- 家庭 Agent 输入框，支持创建提醒、创建任务、切换模式、生成当天摘要。

### 安全定位

- 实时定位展示。
- 今日轨迹列表。
- 守护区域列表和开关。
- 到校、离校、进入危险区等提醒入口。
- 寻找手表。

### 通信与联系人

- 中央语音呼叫和视频呼叫入口。
- 通讯录白名单。
- 拒接陌生人开关。
- 家庭群消息入口。

### 家长管控

- 学习模式、睡眠模式、娱乐模式。
- 应用启用/禁用。
- 应用今日使用时长。
- 聊天风险识别开关。

### 设备与家庭

- 设备绑定。
- 孩子档案。
- 家庭成员管理。
- SIM 号码、话费、流量展示。
- 客服与售后。

### 成长反馈

- 任务激励与星能量。
- 周报框架。
- 运动、睡眠、心情的辅助观察展示。
- Agent 下周行动建议。

## P1 后续增强

- 文件传输、短信代收、通话记录。
- 楼层定位、室内高精定位、路线异常提醒。
- 定时开关机、自动接通、支付限额。
- 应用审批和应用市场。
- 更完整的成长报告模板。
- 基于位置、设备、任务数据的 Agent 主动建议。

## 暂不进入 MVP

- 开放社区、排行榜、活动广场。
- 商城一级入口。
- 付费道具和商业化激励。
- AI 心理诊断或医疗判断。
- 大量娱乐内容池。

## 合规边界

- 定位、轨迹、通讯录、聊天、健康和心情数据都按敏感数据处理。
- 报告中的健康和心情只做辅助观察，不给医疗或心理诊断结论。
- AI 建议必须能解释、能关闭，关键操作必须由家长确认。
- 提供隐私授权、数据留存说明和删除入口。
